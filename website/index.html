HalVM - HAL Volume Manager

About

HalVM is a very simple volume manager daemon which try to mount all removable devices as they are inserted into the system. HalVM uses the ordinary mount command but behaves nicely in collaboration with pmount and pumount, so it's safe to use both on the same system. HalVM logs it's activity to syslog. HalVM is heavily inspired by ivman but is aiming for a much simpler solution. HalVM is writtin in Python using DBus and HAL.

HalVM also comes with a small mount notification utility called HalVM-OSD. HalVM-OSB simply put a small OSD notice of the screen whenever a volume is mounted or unmounted. HalVM-OSD is written in Python and uses HAL, DBus and PyOSD. HalVM-OSD has been developed for use with HalVM but it will work with any volume managers or mounting tool, as it gets informations directly from HAL.

Requirements

The HalVM daemon has the following requirements:

  • Python (tested with Python 2.4)
  • HAL (tested with Hal 0.5.7)
  • DBUS with Python bindings (tested with DBUS 1.0.2 and DBUS-python 0.71)

To use the HalVM-OSD notification tool you also need:

  • PyOSD (tested with PyOSD 0.2.14)
  • libXosd (tested with libXosd 2.2.14)
